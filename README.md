# MATLAB Sudoku Solver UI

This project is a MATLAB App Designer application for solving Sudoku puzzles through a graphical user interface.

## Features

- Interactive 9x9 Sudoku grid
- Input validation
- Automatic puzzle solving
- Clear/reset functionality
- MATLAB App Designer UI

## How to run

1. Download or clone this repository.
2. Open MATLAB.
3. Open `SudokuSolver.mlapp`.
4. Click Run in App Designer.

## Requirements

- MATLAB R2021b or newer
- App Designer
